# neurohackathon

## Install pre-requisites

1. [Install npm and Node.js](https://nodejs.org/).
2. Update npm:
    ```
    # npm install -g npm@latest
    ```
3. Install Polymer CLI:
    ```
    # sudo npm install -g polymer-cli@latest
    ```
    
## Build and serve

Serve:
```
$ npm run serve
```
